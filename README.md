# Zeno5

**Zeno5** is an experimental open-source PlayStation emulator research project focused on understanding and improving modern console emulation through clean architecture, performance optimization, and developer-friendly tooling.

> **Project Status:** Experimental Research Project

## Overview

Zeno5 explores the design and implementation of PlayStation emulation technologies. The project is currently in active development, and many components are experimental. Compatibility, performance, and features will improve over time.

## Features

* Modern C++ codebase
* Cross-platform development
* Experimental PlayStation emulation research
* Performance-focused architecture
* Open-source development

## Installation

### Clone the repository

```bash
git clone https://github.com/tossdace/Zeno5.git
cd Zeno5
```

### Build

Follow the platform-specific build instructions as development progresses. Build documentation will be expanded alongside future releases.

## Usage

After building the project, launch the Zeno5 executable and follow the on-screen instructions.

> **Note:** Zeno5 is under active development. Functionality may be incomplete or unstable.
